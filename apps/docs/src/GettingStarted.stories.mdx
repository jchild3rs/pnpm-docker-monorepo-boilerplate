# Getting Started

## Installation

```bash
pnpm install
```

## Testing

```bash
pnpm test
```

If you wish to test a single file within an app or package, you can run:

```bash
pnpm test --filter api -- "file name regex/path"
```

## Linting

```bash
pnpm lint
```
